# @ulu/mcp-knowledge-base (Development Prototype)

This directory is the temporary development location for the `@ulu/mcp-knowledge-base` package. It is being built and tested within the `@ulu/frontend` repository before being extracted into its own package.

## Purpose

A foundational API and toolkit to expose project structures, conventions, code snippets, and design tokens to AI assistants via the Model Context Protocol (MCP). It allows any project to set up an MCP-compliant "knowledge base".

## Next Steps

We will begin by defining the API schema and documentation in this directory.
