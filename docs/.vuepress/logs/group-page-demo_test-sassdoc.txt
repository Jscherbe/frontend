
# Demo_test-sassdoc



## Mixins




### before-sample() <badge text="mixin" type="tip" vertical="top" />  {#mixin-before-sample} 

  

Mixins can optionally accept a @content block,
passed in brackets. This example creates
a generated element `:before` the element it is used on.
    
    

<SassdocDetails summary="Meta Information">

- **File:** _test-sassdocs.scss
- **Group:** demo_test-sassdoc
- **Type:** mixin
- **Lines (comment range):** 85-104
- **Lines (code):** 105-110


</SassdocDetails>
  

#### Examples

      


``` scss
.item {
  @include samples.before-sample('Test One: ') {
    color: red;
  }
}
```
  
      

#### Parameters


|Name|Type|Description|
|:--|:--|:--|
|$content|`String`|A value for the `content` property|
|$listâ€¦|`arglist`|Sass also allows arglists!|
|@content|`content block`|Any other styles that should be applied to the <code>::before</code> element|

    

#### Mixin Output

A <code>::before</code> pseudo-element,
  with the <code>content</code> property established,
  and any passed-in styled applied.
