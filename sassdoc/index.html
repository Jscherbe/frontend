    variables

    config

    $config: (
  "base":      16px,
  "gap":       0.01em,
  "null-name": "none",
  "default" :  "small"
) !default;

    Description

    Module Settings

    Type

    Map

    Map structure

    Map key NameMap key DescriptionMap key TypeMap key Value
    $config.base

    Assumed pixel base, can change based on users font settings so this is just o get us in the ballpark). Note this is not the base font size but the user agent's or user's browser preferernce. This number is just being used for calculating estimated em sizes from average base. Since pixels are easier to understand but since we allow the user to set their font size. All of our css is relative to that, including most of the layout (rems, other relative units)

    		Number— none
    $config.gap

    The amount to offset min from max media queries incase you are using both (ie prevent overlap)

    		Number— none
    $config.null-name

    The name of the space from 0 to the first breakpoint (doesn't really matter) used when passing breakpoints to JS via content property

    		String— none
    $config.default

    the name of the breakpont that is considered the major change (ie desktop to mobile) used by other modules/components

    		String— none

    sizes

    $sizes: (
  "small"  : calculate.pixel-to-em(680px, get("base")),
  "medium" : calculate.pixel-to-em(1200px, get("base")),

) !default;

    Description

    The default breakpoint sizes (targets are not precise, using em's)
