Default Layout

This is the default layout where the graphic and text sit side-by-side.

Right Aligned Graphic

The graphic sits on the right. Note that the text appears first in the markup for screen reader accessibility.

Top Graphic

The graphic is positioned above the text content and centered.

Align Start

The graphic and text are aligned to the top. This is useful for longer description blocks where a centered icon would feel disconnected from the heading.

Align Baseline

Aligned to the typographic baseline using the `--align-baseline` modifier, paired with `--auto` for natural graphic sizing.

Vertical Separator

A thin separator line is dynamically added between the graphic and the text.

Rule Borders

This layout adds spacing and borders above and below the block.

02

Adaptive Stacking

On wider viewports, this renders side-by-side with a vertical separator. On smaller viewports, it wraps to a column layout, and the separator automatically becomes horizontal.
